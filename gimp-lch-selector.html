L*C*h color selector for GIMP

Download

gimp-lch-selector-0.3.tar.gz

What is this?

A Gimp module that enables to choose the colors in L*C*h color space, which is a polar coordinate representation of L*a*b*.

Install

You don't need to rebuild the Gimp.

tar xvzf gimp-lch-selector.tar.gz
cd gimp-lch-selector
make install

cd ../
