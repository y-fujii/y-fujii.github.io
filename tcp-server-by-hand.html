TCP server by hand

Download

servByHand-0.2.py (要 Python >= 2.5, readline)

これ何

手で応答する TCP サーバ。 telnet のサーバ版みたいな感じ。 telnet でサーバプログラムのテストをすることがあるけど、その逆でクライアントプログラムのテストに使える。

特徴

  • 非同期な実装。
  • クライアントからデータが送信されると即時表示され、かつ入力中の内容を乱さない。
  • 入力に readline を用いているので、送信内容の編集が容易。

バグ

  • 受信したデータに適当に改行が追加される。まあいいや。 → 修正。
